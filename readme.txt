== Description ==

Control your product quantities by adding minimum and maximum quantity rules, step values, default quantities, and more.

== Installation ==

1. Download the plugin from the Order Confirmation page or using the link in your confirmation email.
1. Go to Plugins -> Add New -> Upload, and select the zip file you downloaded.
1. Once installed, click to Activate the plugin.
1. In the dashboard, go to WooCommerce -> Settings -> Products -> Quantity manager and enter your License Key in the box at the top. The license key can be found in your order confirmation email.
1. See the [full documentation](https://barn2.com/kb-categories/wqm-kb/) for further details and instructions.
